## 🛠️ Fitur Utama Aplikasi
1. **Grid Menu Dinamis:** Tampilan menu yang estetik dengan urutan kategori acak/abstrak (selang-seling) untuk pengalaman visual yang lebih organik.
2. **Sistem Pencarian Menu:** Fitur pencarian menu secara *real-time* berdasarkan kata kunci.
3. **Keranjang Belanja & Validasi Stok:** Validasi porsi pesanan agar tidak melebihi stok yang tersedia di database.
4. **Sistem Nota Aman:** Halaman cetak nota transaksi dengan enkripsi parameter URL berbasis Base64.
5. **Autentikasi Admin/Kasir:** Fitur login multi-user untuk masuk ke sistem backend kasir.
